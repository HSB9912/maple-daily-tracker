💰 합산 메소 = (재화수량 × 메소단가)의 합. 가격 미설정 재화는 자동 제외.

📝 구현 요약

  • 데이터 추가: data.prices = { 다조:6800000, 우유:800000, ... }, data.cashRate = 1500 (메소 1억당 원)
  • 설정 탭의 기존 "다조 1개 = X 메소" UI → 모든 재화 자동 나열로 확장 + 현금 시세 박스 추가
  • 정산 표에 💰 합산 메소, 💵 정산 현금 두 컬럼 추가 + 맨 아래 전체 합계 행
  • 매소=1:1 고정 / 경험치=항상 제외 / 가격 비면 합산 제외
  • 모두 Supabase로 자동 동기화
